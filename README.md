# Space Ships
Simple 3d game, player must destroy all meteors!

# Controls
- A / Left arrow- move left
- D / right arrow- move right
- space - shoot

# Supported platforms
- Windows (Visual Studio 2015) 
- Linux (GCC 5.0+ or clang 3,3+)

# Dependencies
- cmake 3.0+
- OpenGL 3.3+
- SDL2
- GLM
- GLEW
- Bullet 2.85
